# Avicenna Take-Home: Notification Lifecycle

## Getting started

1. Clone or download this repository.
2. Read [`CHALLENGE.md`](./CHALLENGE.md) — it has the full task and what to hand back.

## Submitting

When you're done, send us **either**:

- a link to a repository we can access, **or**
- an archive of your project — make sure it includes the `.git/` folder so the commit history comes with it.

Either way, your git history is part of the submission.
